Commands

Cyganworld contains the basic BonziWORLD commands you know and love and likely use! The square brackets indicate what you want to put, please don't include them! This is a list of new/changed commands

God mode

  1. /restart: restarts the server (assumes it automaticaly restarts when it is turned off)
  2. /update: updates the settings, colors and blacklist without the server having to restart. SOME SETTINGS WON'T TAKE AFFECT UNTIL RESTARTED, SUCH AS PORT
  3. /sanitize: now you just do /sanitize and it toggles sanitization on or off.
